Проект выполнен на языке программирования TypeScript с использованием фреймворка NestJS. 
Для установки NestJS необходимо выполнить следующую команду в терминале:

npm i -g @nestjs/cli

В качестве СУБД был использован PostgreSQL. 
Развертывание БД необходимо выполнить, создав базу данных с названием 'REST-API' и 
восстановив её из файла REST-API.sql в формате

В качестве ORM итпользуется Prisma. Добавить её в проект необходимо выполнив команду:

npm i @prisma/client

Для генерации базы данных нужно в файле .env заменить фрагмент строки 
PASSWORD на пароль вашего пользователя postgres.

Также в проекте используется движек представлений EJS.
Для его установки в терминал нужно ввести команду:

npm install ejs --save

После чего выполнить команду 'npm run start:dev'
